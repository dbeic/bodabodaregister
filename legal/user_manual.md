## 4. MEMBER REGISTRATION

### 4.1 How to Register a Member

**Step 1:** Click "Register" in the navigation menu

**Step 2:** Fill in member details:
- **Member Number:** Unique identifier (e.g., 001, 002)
- **Full Name:** Complete legal name
- **National ID:** Valid Kenyan ID number
- **Telephone:** Active phone number
- **Group/Stage:** Operating location
- **Chairman Name:** Group chairman's name
- **Chairman Phone:** Chairman's phone number
- **Motorcycle Registration:** Vehicle registration plate
- **Next of Kin Name:** Emergency contact name
- **Next of Kin Phone:** Emergency contact phone

**Step 3:** Upload passport photo
- Click "Choose File" under Passport Photo
- Select a JPG, JPEG, or PNG image
- Photo should be clear and passport-style

**Step 4:** Click "Register" to submit

**Step 5:** Wait for badge generation
- System will generate QR code
- System will generate professional badge
- Member is registered successfully!

## 5. BADGE MANAGEMENT

### 5.1 Viewing a Badge

**Option A:** Click on member name from dashboard
**Option B:** Search for member and click "View Badge"

### 5.2 Badge Actions

**Download PNG:**
- Click "Download PNG"
- Badge downloads as image file
- Suitable for digital use

**Download PDF:**
- Click "Download PDF"
- Professional PDF for printing
- High quality, print-ready

**With Bleed:**
- Click "With Bleed" option
- Includes bleed area for professional printing
- Crop marks included

**Print:**
- Click "Print" button
- Use browser print functionality
- Optimized for physical printing

**Regenerate:**
- Click "Regenerate" button
- Creates a fresh badge image
- Use if current badge is corrupted

### 5.3 Badge Status

The badge shows:
- **Member Number:** Unique identifier
- **Full Name:** Member's complete name
- **National ID:** ID number
- **Phone:** Contact number
- **Motorcycle:** Registration plate
- **Group:** Operating location
- **Chairman:** Group chairman
- **QR Code:** Scannable verification code
- **Date Issued:** Registration date
- **Validity:** Valid until December 31, 2027

## 8. BATCH BADGE ISSUANCE

### 8.1 When to Use Batch Issuance
- New member onboarding
- Mass badge generation
- Replacing lost badges

### 8.2 How to Issue Batch Badges
1. Click "Batch Issue" in navigation
2. Select print format:
   - PNG only
   - PDF only
   - Both PNG and PDF
3. Toggle bleed area (for professional printing)
4. Select print quality:
   - Draft (lower quality, faster)
   - Standard (medium quality)
   - High (best quality)
5. Click "Generate Badges"
6. Download the ZIP file
7. Extract and print badges

### 8.3 Important Notes
- Only unissued members are included
- Existing badges are not regenerated
- ZIP file contains all selected formats
- Files are named by member number

## 10. TROUBLESHOOTING

### 10.1 Common Issues and Solutions

**Cannot Login**
- Check username and password
- Verify internet connection
- Contact administrator for password reset

**Badge Not Generating**
- Check member data completeness
- Verify photo upload
- Click "Regenerate" button
- Contact support

**QR Code Not Scanning**
- Ensure QR code is clear
- Use appropriate scanning app
- Try different lighting
- Regenerate badge

**Photo Upload Fails**
- Check file size (max 16MB)
- Use JPG, JPEG, or PNG format
- Ensure image is not corrupt

**Slow Loading**
- Check internet connection
- Reduce search results (be more specific)
- Try using incognito/private mode

### 10.2 Error Messages

| Error | Solution |
|-------|----------|
| "Member number already exists" | Use unique member number |
| "National ID already registered" | Member already in system |
| "Invalid photo file" | Use JPG, JPEG, or PNG |
| "Badge file not found" | Regenerate badge |
| "Database connection error" | Contact support |
